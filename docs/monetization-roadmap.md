- `web/index.html` 의 애드센스 스니펫 `ca-pub-XXXX` → 발급받은 `ca-pub-실제번호`
- `web/public/ads.txt` 의 `pub-0000…` → 같은 발급 번호
- `web/index.html` 의 Cloudflare Web Analytics `REPLACE_WITH_YOUR_CF_TOKEN` → CF 대시보드에서 발급한 토큰
  - (애널리틱스는 애드센스와 무관 — 트래픽 출처를 보려면 지금 바로 교체 가능)

> ⚠️ 시크릿/토큰을 깃에 커밋하지 말 것(저장소 공개). 단, 위 셋은 클라이언트 공개 ID라 index.html에 들어가는 게 정상.

## 3. 성장 로드맵 (수익의 진짜 동력)

**다음 (성장 핵심):**
- **이메일 가격 알림** — 찜한 게임이 역대최저면 메일. 무료(Cloudflare Email Workers / Resend 무료티어)로 서버비 0 유지. 정보통신망법/GDPR 동의·옵트아웃 함께 설계. (업계 1등 트래픽의 60~65%가 '알림 받고 다시 온 사람')
- **커뮤니티 유입** — 주 1회 "이번 주 역대최저 갱신" 큐레이션 글을 아카라이브 핫딜·루리웹·퀘이사존에 정보 형식으로(광고 X). 신규 도메인의 현실적 초기 유입 + 백링크.

**나중:**
- **언어별 URL(/en/·/ja/ 등) + hreflang** — 구글이 6개 언어 각각 색인. 영어권 트래픽 비중↑ → RPM↑, 프리미엄 광고망 진입 발판. Cloudflare Pages 정적 라우팅으로 무료 구현.
- **공식 스토어 제휴(키샵 아님)** — 스팀 appid를 GOG(6%)·Humble(5%) 상품과 매칭(D1 매칭 테이블), 해당 게임이 거기 있을 때만 "공식 스토어로 보기" 보조 링크. `rel="sponsored"` + "제휴 링크로 수수료를 받을 수 있습니다" 고지 필수.
- **프리미엄 광고망** — 트래픽 월 수만 PV↑ 후 Ezoic→Mediavine(월 1만 세션)→Raptive(월 2.5만 PV+영어권 50%) 순. 같은 트래픽에서 RPM↑.
- **다이렉트게임즈(H2 Interactive)** — 한국·원화 친화로 사용자층과 가장 맞음. 공개 제휴 없어 직접 문의 필요.

## 4. 수익원 우선순위

1. **구글 애드센스** (적합도 高, 노력 S) — 1차 수익원. 단가 낮아 트래픽 규모 의존(월 10만 PV대부터 의미).
2. **후원 버튼**(GitHub Sponsors/Patreon, 적합도 高, 노력 S) — 금액보다 '광고 없이도 버티는 신뢰' 이미지 + 수익원 분산. About/푸터에 한 줄.
3. **공식 스토어 제휴 GOG·Humble** (적합도 高, 노력 M) — 정직 브랜드와 충돌 최소. 단가는 보조 수준.
4. (후순위) Fanatical·GMG — 공식 리셀러지만 '키샵 미는 사이트' 오해 위험, 명확히 구분 표기해야.

## 5. 절대 하지 말 것

- **키샵(G2A·Kinguin·Eneba) 제휴 금지** — 불법은 아니나 도난키·차지백으로 개발사 피해. "키샵 없음·정직" 브랜드를 스스로 파괴(=금연 캠페인하며 담배 파는 격). 수익은 푼돈(1만원 키 5%=500원), 잃는 신뢰는 회복 불가.
- **스팀 자체 제휴 기대 금지** — 밸브는 공개 제휴 프로그램 없음. 스팀 직링크론 0원.
- **네이버 애드포스트** — 자체 도메인엔 적용 불가(2015년 외부 지원 종료). 한국 디스플레이 광고는 애드센스 단독.
- **쿠팡파트너스 등 무관 제휴** — 주제 어긋나 신뢰·UX 훼손 + 애드센스 정책 마찰.
- **CheapShark 약관 위반 금지** — 크롤러가 CheapShark 데이터를 씀. 자기 제휴 링크로 우회하면 데이터 소스가 끊길 수 있음(필요 시 admin@cheapshark.com에 확인).
- **애드센스 단일 의존 금지** — 계정 정지 한 방에 수익 0. 광고+후원 최소 2채널 분산.
- **광고를 영수증 UI에 욱여넣기 금지** — 폴드당 3개·데스크탑 5~7개 이내, 뷰포트 15% 초과 금지.

## 6. 빈약 페이지 noindex 기준 (2026-06-20 적용)

- 색인 조건: `현재가 > 0` AND (`리뷰 수 ≥ 300` OR `메타크리틱 > 0`).
- 위치: `web/src/lib/head.js`(클라), `web/functions/game/[appid].js`(봇), `web/functions/sitemap.xml.js`(사이트맵) — 셋 다 같은 기준.
- 튜닝: 데이터가 성숙해 정상 게임이 충분히 쌓이면 `300`을 낮춰 색인 범위를 넓혀도 됨(현 양산형 인디 리뷰가 100~150대라 300으로 잡음).
